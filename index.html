بداية تعلمي في GitHub

مرحباً! هذه الصفحة توثق أولى خطواتي في تعلم منصة GitHub.

لماذا بدأت أتعلم GitHub؟

كنت أسمع كثيراً عن Git و GitHub كأدوات يستخدمها المطورون لإدارة المشاريع وتتبع التعديلات. فقررت أن أبدأ بتعلم الأساسيات حتى أتمكن من استخدامها في مشاريعي الشخصية والمستقبلية.

ما الذي تعلمته حتى الآن؟

  • إنشاء حساب على GitHub.
  • فهم الفرق بين Git و GitHub.
  • إنشاء مستودع (Repository) جديد.
  • رفع الملفات باستخدام Git Bash.
  • كتابة ملف README.md.
  • استخدام الأوامر الأساسية مثل git init و git commit.

أين أذهب من هنا؟

أخطط لتعلم المزيد عن:

  • Git branches
  • حل تعارضات الدمج (Merge Conflicts)
  • المساهمة في مشاريع مفتوحة المصدر
